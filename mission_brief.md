# Mission Brief: SEO4Coaches Website

## Project Overview
This project is a 4-page premium portfolio website for SEO4Coaches (Jeffrey Pendon). It targets coaches and therapists with SEO and automation services.

## Main Components
- `index.html`: Home page with conversion-focused sections.
- `services.html`: Pricing packages and specific service offers.
- `about.html`: Personal bio and ethical SEO philosophy.
- `contact.html`: Optimized inquiry form.
- `style.css`: Unified Sage Green and Soft Blue design system.
- `assets/`: Hero images and profile photos.

## Tech Stack
- HTML5
- Vanilla CSS
- (Assumed) Hostinger for deployment.

Open the app on localhost for local development and previewing.
